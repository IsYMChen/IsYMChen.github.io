Hexo for Ubuntu

发表于 2019-01-26 | 分类于 一些工具的安装及配置

前提

这个Hexo的安装与配置是在阿里云轻量应用服务器Ubuntu16.04的环境下实操的

1. 使软件包列表最新

$ sudo apt-get update

注：

  • 如果发现之后有文件无法下载的情况，那就换个更新源吧，下面给出一个更新源参考地址
    https://www.cnblogs.com/bovenson/p/5752213.html
    换好之后再执行$ sudo apt-get update一下

2. 安装git

$ sudo apt-get install git

3. 安装nvm

$ git clone https://github.com/creationix/nvm.git ~/.nvm && cd ~/.nvm && git checkoutgit describe –abbrev=0 –tags

4. 配置nvm

$ sudo vi ~/.bashrc

在最后一行添加

$ source ~/.bashrc 使配置生效

5. 安装node.js

$ wget -qO- https://raw.github.com/creationix/nvm/v0.33.11/install.sh | sh
$ nvm install stable

6. 安装Hexo

$ npm install -g hexo-cli

7. 创建你自己的博客框架

$ mkdir YourBlogName
$ hexo init YourBlogName
$ cd YourBlogName
$ npm install
$ hexo server

到这步为止，你创建的博客已经可以在本地进行访问，输入http://localhost:4000就可以进行访问

附上一个中文文档地址

https://hexo.io/zh-cn/docs/
